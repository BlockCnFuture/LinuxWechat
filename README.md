## LinuxWechat
This project was modified from [freechat](https://github.com/eNkru/freechat),fixed the login problem. 

## Usage
```js
npm install electron
npm start
```
## Compile
```js
npm run dist:linux
npm run dist:win
npm run dist:dist:macOS
```
