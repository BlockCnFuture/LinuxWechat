## LinuxWechat
此项目修改自 [freechat](https://github.com/eNkru/freechat),修复了登录问题。  
  
## 用法
```js
npm install electron
npm start
```
## 编译
```js
npm run dist:linux
npm run dist:win
npm run dist:macOS
```
